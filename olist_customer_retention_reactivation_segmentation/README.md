# Olist Customer Retention Analysis

## Overview
This project analyzes customer purchasing behavior using the Olist e-commerce dataset. The goal is to identify high-value customers and evaluate recent purchasing activity by combining lifetime spending with recent order frequency.

The analysis demonstrates how SQL can transform transactional order data into customer-level insights useful for retention and marketing strategies.

---

## Business Question
Which customers generate the most lifetime revenue, and how recently have they purchased?

Understanding this helps businesses:
- Identify high-value customers
- Track recent engagement
- Prioritize retention and loyalty strategies

---

## Dataset
The analysis uses the **Olist Brazilian E-commerce dataset**.

Key tables used:

- `orders` – order-level purchase information
- `order_payments` – payment values for each order

---

## Method

The SQL query builds customer-level metrics using two main steps:

### 1. Lifetime Spend
Calculate total revenue generated by each customer across all orders.

### 2. Recent Activity
Measure recent purchasing behavior by:
- Counting orders in the last 100 days
- Identifying the most recent purchase date

These metrics are combined to identify **high-value customers who remain active**.

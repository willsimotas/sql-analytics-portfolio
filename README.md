# SQL Analytics Portfolio

This repository contains SQL-based analytics case studies built using PostgreSQL.  
The projects focus on revenue analysis, customer segmentation, and business performance insights.

---

## About

I built this portfolio to demonstrate how I approach business problems using SQL — from defining the objective to structuring queries with clean logic, CTEs, and aggregation strategies.

---

## Projects

### 1. High-Value Active Customers
**Objective:**  
Identify customers with lifetime spend greater than $1,000 who have also made a purchase within the last 100 days.

**Key Concepts:**
- Common Table Expressions (CTEs)
- Lifetime value aggregation
- Recent activity filtering
- Join logic across multiple tables

→ See `customer_segmentation/` for full query and documentation.
